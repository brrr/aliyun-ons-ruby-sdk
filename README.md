# Ons

unoffical aliyun ONS sdk (ruby version)

## Requirements

* Linux/Unix
* Ruby 2.1.5 or later
* Boost C++ library

## Installation

### install Ruby

```sh
$ rvm install 2.1.5
```

### install Boost C++ library

```sh
$ apt-get install libboost-all-dev
```

### install ons

```sh
$ git clone https://github.com/souche/aliyun-ons-ruby-sdk.git
$ cd ons
$ ./bin/setup
$ ./bin/rake install
```
